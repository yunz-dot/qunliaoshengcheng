API 魔法设置
接口格式兼容 /v1/chat/completions。浏览器直连接口可能遇到 CORS 限制；若报错，需要代理服务或后端转发。
提示：Key 放前端有泄露风险。建议只在本地使用，不要把带 Key 的网页公开分享。
① 群聊基础设定 核心
自动匹配会根据群聊主题、User人设、目标和群聊钩子来勾选公共标签，并按角色数量为每个角色分配独立分层标签；没配置 API 时会用本地逻辑兜底。
② 群聊世界观 / 场域 清晰分层
按“世界是什么、规则怎么运转、日常在哪里发生、关系是什么味、剧情为什么推进”来选，避免把职业、关系梗和场景混在同一层。
③ 角色卡公共输出区块
这里只控制每张角色卡需要输出哪些栏目。身份类型、能力、人格、关系、说话风格都已移动到“单个角色独立分层设定”里，每个角色单独配置，避免重复。
④ 单个角色独立分层设定 每人单独选
每个角色都在这里单独选择身份类型、能力、人格、关系、化学反应和发言机制。①里的AI智能匹配会按角色功能互补、关系化学反应和User人设，为每个人分配独立标签，你也可以手动微调。
⑤ 专业知识 / 输出控制
导出文件
导出当前生成结果，或导出/导入全部存档。
生成结果
结果会优先输出群聊世界观、成员关系网、逐个角色分层卡、群聊发言规则和沉浸式群聊压缩导入版。
点击底部「✨ 生成群聊角色」。
